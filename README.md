#  Wordpress Skeleton Theme

A base wordpress theme to help improve development workflow.

## Features

<ul>
	<li>Basic Responsive Styles</li>
	<li>CSS Reset</li>
	<li>CSS + JS Concatenation + Minification</li>
	<li>CSS + JS Linter</li>
	<li>jQuery</li>
	<li>Modernizr</li>
	<li>Sass Support + Included Mixins</li>
</ul>
